# BITStack
A University Stack Web App

## Installation

1. Install requirements

```
pip3 install -r requirements.txt
```
2. Make migrations
```
python3 manage.py makemigrations
```
3. Migrate
```
python3 manage.py migrate
```
4. Run server
```
python3 manage.py runserver
```
5. Open app in browser
```
http://127.0.0.1:8000/
```
